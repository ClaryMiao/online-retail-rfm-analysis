# Online Retail Customer Segmentation and RFM Analysis

## Project Overview

This project analyzes customer purchasing behavior using the Online Retail Dataset and applies RFM Analysis and K-Means Clustering to identify different customer segments.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Project Workflow

### Data Cleaning

- Removed missing Customer IDs
- Removed duplicate records
- Removed cancelled transactions

### RFM Analysis

Calculated:

- Recency
- Frequency
- Monetary Value

for each customer.

### Customer Segmentation

Customers were segmented into groups based on purchasing behavior.

### K-Means Clustering

Applied clustering techniques to identify customer groups with similar characteristics.
